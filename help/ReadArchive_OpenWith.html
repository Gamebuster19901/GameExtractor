Open Archive with Plugin

Back to Index

Opens an archive on your computer, using a specific plugin.

A list of files and directories will be displayed in the Side Panel. At the bottom of the Side Panel, you can choose a specific plugin that you want to use. When you choose a plugin, the list of files will change, to only show files that might be compatible with that plugin. To open a file, you can either double-click a file in the list, or choose a file and click the Open Archive button at the bottom. Game Extractor will try to open the file using the plugin you have chosen.
