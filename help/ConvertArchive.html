Convert Archive

Back to Index

Saves an archive to your computer, in a specific format. Can be used to convert an archive into a different format, such as by changing a proprietary archive into a ZIP file.

A list of files and directories will be displayed in the Side Panel. At the bottom of the Side Panel, choose the format for the new archive, enter a filename, and click the Convert Archive button. Game Extractor will save the archive in the format you have chosen.


This is the button to Convert an Archive

This function can only be used on archives that support editing.
This function is only available in the Full Version of Game Extractor
