Options

Back to Index

Allows you to set a variety of Game Extractor properties. Choose a option group from the box at the top, and the side panel will change to show all the options in that group. When you have finished changing the options, click the Save Options button at the bottom.


This is the button to open the Options

The options are categorized as follows:

Appearance: Change the color and appearance of the program.
Directories: The directories used for various things such as exporting and saving.
General: Basic options that don't fit in to any other category.
Hex Viewer Fields: Choose the representations that are shown in the Hex Viewer when a cell is selected.
Opening and Saving Archives: Settings for Opening and Saving archives, and for affecting the way Game Extractor determines the correct plugin for opening an archive.
Popups: Enable or disable the popups that appear when performing a task.
Previewing Files: Settings for Previewing files in an archive.
Scanners: Choose the scanners that will be used when Scanning for files in an archive.
Table Columns: Choose the columns that are shown in the File List Table.
Toolbar Buttons: Change and re-order the buttons that are shown on the toolbar.

If you move the mouse over an option, a more details description of the option will be shown in the statusbar at the bottom of the screen.
