External Software

Back to Index

Game Extractor can interface with other software to help perform certain tasks. The software below is all optional, but if you install them and configure them as described below, Game Extractor will be more powerful. All the software on this page is free to download and use.

FFMPEG

Game Extractor uses this to preview some additional audio formats such as *.xwma and Xbox sound files.

Download FFMPEG from https://ffmpeg.org/
Install it to your computer
Open Game Extractor and go to the Options (menu > Tools > Options)
In the Options Side Panel, choose the option group called Previewing Files
Enter the directory where you installed FFMPEG, or click the Choose button and find it on your PC
Click the Save Options button


QuickBMS

Game Extractor uses this to decompress some archives, and for opening archives using BMS scripts.

Download QuickBMS from http://aluigi.altervista.org/papers/quickbms.zip
Unzip it to your computer
Open Game Extractor and go to the Options (menu > Tools > Options)
In the Options Side Panel, choose the option group called Opening and Saving Archives
Enter the directory where you installed QuickBMS, or click the Choose button and find it on your PC
Click the Save Options button
