Previews and Thumbnails for Image Files on your PC

Back to Index

Game Extractor can be used to display thumbnails and previews for image files on you computer. This can be incredibly useful for games which have some images stored as separate files instead of being contained within an archive.

Click the New Archive button at the top of the screen.
Click the Add Files button at the top of the screen, which will open a list of files and directories in the Side Panel.
Select the image files on your computer, and drag them into the File List (or click the Add Files button).
Click the Thumbnail View button at the top of the screen, to display the images as thumbnails.

The end result is that all the images can be viewed in the Thumbnail View, or Previewed within Game Extractor. Furthermore, you can Save all these files as an archive, or even Convert all the images to a different format.

These functions are only available in the Full Version of Game Extractor
