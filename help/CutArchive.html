Cut Archive

Back to Index

Copies parts of an archive for analysis.

A list of files and directories will be displayed in the Side Panel. To cut a file, select a file to cut, enter a name for the new cut file, choose the amount of data to cut, and click the Cut Archive button at the bottom. Game Extractor will copy some of the data from this file, and zip it up for you. This zip file can then be sent to an analyst to examine, in the hope of writing a script or plugin to support the game.
