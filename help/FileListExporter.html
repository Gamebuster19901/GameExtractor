File List Exporter

Saves the list of files to your computer.

In the box at the top, select the type of document that will be used to save the File List. Use the checkboxes under it to select the types of file data that you want to save. Pressing the Save File List button will create a file that lists all the files in the archive. The document is saved to the directory specified in the Options.
