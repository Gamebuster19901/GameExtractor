Undo

Back to Index

Reverses a change that you made to an archive.

Some tasks, such as Adding, Removing, Renaming, and Replacing files can be reversed if you make a mistake. Clicking an item in this menu will reverse the change that was done. Clicking an item lower in the menu will undo all tasks before it too. If a task is undone, you can redo it using the Redo menu.
