Previews the selected file in the File List.

A preview of the selected file will be displayed in the Side Panel. You can also show a preview quickly by double-clicking a file in the File List. Many common image and audio files can be previewed - if a preview can't be shown then the file will be loaded in the Hex Viewer instead.

Some previews can be converted into different formats. To do this, choose a new format in the box at the bottom, and click the Convert button. The converted file will have the same name as the original file, and will be saved to the preview directory specified in the Options.
