Hex Viewer

Back to Index

Shows a Hex Viewer representation of the selected file in the File List.

The selected file will be displayed in the Side Panel, as a table of raw data. The hex viewer can be configured to show the file as either byte, hex, or character representations via the buttons at the top. Selecting a cell in the table will display various representations of those bytes of the file - the representations shown can be turned on or off in the Options.


This is the button to show a file in the Hex Viewer
