Add Files

Back to Index

Adds files to the current archive.

A list of files and directories will be displayed in the Side Panel. To add files to an archive, select any number of files and directories, and click the Add Files button at the bottom. Game Extractor will add all the selected files to the current archive. If you have any directories selected, it will add all the files in those directories too.

You can also add files by selecting and dragging files from the Side Panel into the File List, then choosing the Add Files option.
