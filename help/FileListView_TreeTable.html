Group Table

Back to Index

This File List view consists of 2 parts - the tree on the left, and the table on the right. Above the tree is a box that allows you to choose how the files are grouped. By choosing a group, the tree is changed to show all the different values for the particular group. Clicking an item in the tree will display all matching files in the table on the right. If you click the Archive item in the tree, all files in the archive are shown.

For example, if you choose to group by Extension, the tree will display a list of all the extensions used in the archive. Clicking on a particular extension in the tree will display only the files that have that extension.

The columns shown in the table can be turned on and off in the Options. Clicking on the table heading will sort the files by the chosen column.
