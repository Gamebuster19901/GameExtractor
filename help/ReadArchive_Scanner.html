Scan Archive

Back to Index

Scans an archive file on your computer for known file types.

A list of files and directories will be displayed in the Side Panel. To scan a file, you can either double-click a file in the list, or choose a file and click the Scan Archive button at the bottom. Game Extractor will scan through the file you have selected, looking for common files such as audio and images. The scanners that are used can be turned on and off in the Options.
